# Mend Phase 1 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship ctscan's `--format json` output mode and the Mend.app Phase 1 — a native macOS SwiftUI app with Menu Bar, Overview tab, and Scan tab that reads from the bundled ctscan binary.

**Architecture:** ctscan gains a `--format json` flag that emits a single JSON object (one key per module with status/value/message). Mend is a new SwiftUI repo that bundles the ctscan binary, spawns it via `Process` API, decodes the JSON into Swift models, persists scan history to `~/.mend/history.json`, and renders the result as an Overview dashboard and a detailed Scan tab. An `NSStatusItem` menu bar icon provides always-on background scanning with warning badges.

**Tech Stack:** Bash 3.2 (ctscan), SwiftUI + AppKit (Mend), XCTest, Xcode 15+, macOS 13.0 deployment target.

**Phases 2 and 3** (App Remover, Cache Cleaner, Updates, Startup, Storage) are separate follow-on plans. This plan produces a shippable v1.0.

```bash
FORMAT="text"

# ── JSON result collection ──────────────────────
declare -A _JSON_STATUS=()
declare -A _JSON_VALUE=()
declare -A _JSON_MSG=()

json_result() {
  # Usage: json_result <module> <status> <value> <message>
  # status: ok | warn | error | unavailable
  local mod="$1" status="$2" value="$3" message="${4:-}"
  _JSON_STATUS["$mod"]="$status"
  _JSON_VALUE["$mod"]="${value//\"/\'}"   # replace " with ' to keep JSON valid
  _JSON_MSG["$mod"]="${message//\"/\'}"
}

emit_json() {
  local first=true
  printf '{\n'
  for mod in "${ALL_MODULES[@]}"; do
    [[ -z "${_JSON_STATUS[$mod]+x}" ]] && continue
    $first || printf ',\n'
    printf '  "%s": {"status": "%s", "value": "%s", "message": "%s"}' \
      "$mod" "${_JSON_STATUS[$mod]}" "${_JSON_VALUE[$mod]}" "${_JSON_MSG[$mod]}"
    first=false
  done
  printf '\n}\n'
}
```

# ── JSON format tests ───────────────────────────────────────────────────────

test_json_output_is_valid_json() {
  local out
  out=$(bash bin/ctscan --format json 2>/dev/null)
  if ! echo "$out" | python3 -m json.tool > /dev/null 2>&1; then
    echo "FAIL: --format json did not produce valid JSON"
    echo "Output was: $out"
    return 1
  fi
  echo "PASS: --format json produces valid JSON"
}

test_json_contains_all_modules() {
  local out
  out=$(bash bin/ctscan --format json 2>/dev/null)
  local missing=()
  for mod in identity brew agents storage ssd battery thermal memory processes docker timemachine updates wifi security; do
    if ! echo "$out" | python3 -c "import sys,json; d=json.load(sys.stdin); assert '$mod' in d" 2>/dev/null; then
      missing+=("$mod")
    fi
  done
  if [[ ${#missing[@]} -gt 0 ]]; then
    echo "FAIL: JSON missing modules: ${missing[*]}"
    return 1
  fi
  echo "PASS: JSON contains all 14 module keys"
}

test_json_each_module_has_required_fields() {
  local out
  out=$(bash bin/ctscan --format json 2>/dev/null)
  local failed=false
  for mod in identity brew battery thermal memory; do
    if ! echo "$out" | python3 -c "
import sys, json
d = json.load(sys.stdin)
m = d['$mod']
assert 'status' in m, 'missing status'
assert 'value' in m, 'missing value'
assert 'message' in m, 'missing message'
assert m['status'] in ('ok','warn','error','unavailable'), 'invalid status: '+m['status']
" 2>/dev/null; then
      echo "FAIL: module '$mod' missing required fields or invalid status"
      failed=true
    fi
  done
  $failed && return 1
  echo "PASS: sampled modules have required fields with valid status values"
}

test_json_no_text_output() {
  local out
  out=$(bash bin/ctscan --format json 2>/dev/null)
  # Should not contain ANSI escape codes or ctscan header text
  if echo "$out" | grep -qE '(══|✓|⚠|·|\\033\[)'; then
    echo "FAIL: --format json output contains text/ANSI formatting"
    return 1
  fi
  echo "PASS: --format json output contains no text formatting"
}

test_json_single_module() {
  local out
  out=$(bash bin/ctscan --format json battery 2>/dev/null)
  local count
  count=$(echo "$out" | python3 -c "import sys,json; print(len(json.load(sys.stdin)))" 2>/dev/null)
  if [[ "$count" != "1" ]]; then
    echo "FAIL: single-module JSON should have 1 key, got: $count"
    return 1
  fi
  echo "PASS: single-module JSON contains exactly 1 key"
}

# Run JSON tests
test_json_output_is_valid_json   || FAIL_COUNT=$((FAIL_COUNT+1))
test_json_contains_all_modules   || FAIL_COUNT=$((FAIL_COUNT+1))
test_json_each_module_has_required_fields || FAIL_COUNT=$((FAIL_COUNT+1))
test_json_no_text_output         || FAIL_COUNT=$((FAIL_COUNT+1))
test_json_single_module          || FAIL_COUNT=$((FAIL_COUNT+1))
```

```swift
import SwiftUI

// MARK: - Wire types

enum ModuleStatus: String, Codable {
    case ok
    case warn
    case error
    case unavailable

    var color: Color {
        switch self {
        case .ok:          return .green
        case .warn:        return .orange
        case .error:       return .red
        case .unavailable: return Color(.systemGray)
        }
    }

    var badgeSymbol: String {
        switch self {
        case .ok:          return "checkmark.circle.fill"
        case .warn:        return "exclamationmark.triangle.fill"
        case .error:       return "xmark.circle.fill"
        case .unavailable: return "minus.circle.fill"
        }
    }
}

struct ModuleResult: Codable, Equatable {
    let status: ModuleStatus
    let value: String
    let message: String
}

/// The full output of one ctscan run — keyed by module name.
typealias ScanResults = [String: ModuleResult]

/// One persisted scan entry in history.json.
struct ScanSnapshot: Codable {
    let timestamp: Date
    let results: ScanResults
}

// MARK: - Display helpers

extension ScanResults {
    /// Worst status across all modules. nil if empty.
    var overallStatus: ModuleStatus? {
        if values.contains(where: { $0.status == .error })       { return .error }
        if values.contains(where: { $0.status == .warn })        { return .warn }
        if values.contains(where: { $0.status == .ok })          { return .ok }
        return .unavailable
    }
}
```

```swift
import Foundation

enum CtscanError: LocalizedError {
    case binaryNotFound
    case processFailure(exitCode: Int32, stderr: String)
    case decodingFailure(underlying: Error, output: String)

    var errorDescription: String? {
        switch self {
        case .binaryNotFound:
            return "ctscan binary not found in app bundle."
        case .processFailure(let code, let stderr):
            return "ctscan exited with code \(code): \(stderr)"
        case .decodingFailure(_, let output):
            return "Could not parse ctscan output. Raw: \(output.prefix(200))"
        }
    }
}

actor CtscanRunner {

    static let shared = CtscanRunner()

    /// Runs the bundled ctscan binary with --format json and returns decoded results.
    func run(modules: [String] = []) async throws -> ScanResults {
        guard let binaryURL = Bundle.main.url(forResource: "ctscan", withExtension: nil) else {
            throw CtscanError.binaryNotFound
        }

        let process = Process()
        process.executableURL = binaryURL
        var args = ["--format", "json"]
        args.append(contentsOf: modules)
        process.arguments = args

        let stdoutPipe = Pipe()
        let stderrPipe = Pipe()
        process.standardOutput = stdoutPipe
        process.standardError  = stderrPipe

        try process.run()
        process.waitUntilExit()

        let stdoutData = stdoutPipe.fileHandleForReading.readDataToEndOfFile()
        let stderrData = stderrPipe.fileHandleForReading.readDataToEndOfFile()

        if process.terminationStatus != 0 {
            let stderr = String(data: stderrData, encoding: .utf8) ?? ""
            throw CtscanError.processFailure(exitCode: process.terminationStatus, stderr: stderr)
        }

        let decoder = JSONDecoder()
        do {
            return try decoder.decode(ScanResults.self, from: stdoutData)
        } catch {
            let raw = String(data: stdoutData, encoding: .utf8) ?? "<non-utf8>"
            throw CtscanError.decodingFailure(underlying: error, output: raw)
        }
    }
}

import Foundation

/// Persists scan snapshots to ~/.mend/history.json.
/// Max 90 entries; oldest are pruned on each save.
actor HistoryStore {

    static let shared = HistoryStore()

    private let fileURL: URL
    private let maxEntries = 90

    private lazy var decoder: JSONDecoder = {
        let d = JSONDecoder()
        d.dateDecodingStrategy = .iso8601
        return d
    }()

    private lazy var encoder: JSONEncoder = {
        let e = JSONEncoder()
        e.dateEncodingStrategy = .iso8601
        return e
    }()

    init(fileURL: URL? = nil) {
        if let url = fileURL {
            self.fileURL = url
        } else {
            let home = FileManager.default.homeDirectoryForCurrentUser
            let dir  = home.appendingPathComponent(".mend")
            try? FileManager.default.createDirectory(at: dir, withIntermediateDirectories: true)
            self.fileURL = dir.appendingPathComponent("history.json")
        }
    }

    /// Prepends a new snapshot and trims to maxEntries. Creates the file if absent.
    func save(_ results: ScanResults) throws {
        var history = (try? load()) ?? []
        history.insert(ScanSnapshot(timestamp: Date(), results: results), at: 0)
        if history.count > maxEntries {
            history = Array(history.prefix(maxEntries))
        }
        let data = try encoder.encode(history)
        try data.write(to: fileURL, options: .atomic)
    }

    /// Returns all stored snapshots, newest first.
    func load() throws -> [ScanSnapshot] {
        let data = try Data(contentsOf: fileURL)
        return try decoder.decode([ScanSnapshot].self, from: data)
    }

    /// Returns the snapshot immediately before the most recent one, for delta computation.
    func previousSnapshot() -> ScanSnapshot? {
        return (try? load())?.dropFirst().first
    }
}

import SwiftUI

@MainActor
final class AppState: ObservableObject {

    static let shared = AppState()

    @Published var results: ScanResults = [:]
    @Published var isScanning: Bool = false
    @Published var lastScanned: Date? = nil
    @Published var scanError: String? = nil

    /// The snapshot just before the most recent scan — used for delta display.
    var previousResults: ScanResults? = nil

    func scan() async {
        guard !isScanning else { return }
        isScanning = true
        scanError = nil

        do {
            // Load previous before overwriting
            previousResults = await HistoryStore.shared.previousSnapshot()?.results

            let fresh = try await CtscanRunner.shared.run()
            results = fresh
            lastScanned = Date()
            try await HistoryStore.shared.save(fresh)
        } catch {
            scanError = error.localizedDescription
        }

        isScanning = false
    }

    /// Returns the delta string for a module value if history exists.
    /// e.g. "↓0.8% since last scan" or nil
    func delta(for module: String) -> String? {
        guard
            let current  = results[module]?.value,
            let previous = previousResults?[module]?.value,
            current != previous
        else { return nil }
        return "\(previous) → \(current)"
    }
}

# Mend

A native macOS maintenance app. Scan your system health, clean caches, remove apps, and manage updates — all in one place.

## Install

```bash
brew install --cask mend
```

## Requirements

- macOS 13.0 (Ventura) or later
- Apple Silicon or Intel

## Development

**Prerequisites:** Xcode 15+

```bash
git clone https://github.com/onurtellioglu/mend
open Mend.xcodeproj
```

Build with ⌘B, run with ⌘R, test with ⌘U.

The `Resources/ctscan` binary is the bundled [ctscan](https://github.com/onurtellioglu/ctscan) diagnostic engine. Update it by copying a new build from the ctscan repo.

## Phase 1 (current)

- Menu Bar icon with health warning badge
- Overview tab — metric cards for 9 key health indicators with history deltas
- Scan tab — full 14-module health report

## Roadmap

- Phase 2: App Remover, Cache Cleaner
- Phase 3: Updates, Startup Manager, Storage Analyzer
```
