about

Your Mac's health report.
No daemon. No GUI. No BS.

ctscan runs 14 diagnostic checks from the terminal and gives you a clean, color-coded report in less than a second. Battery health, memory pressure, disk status, security posture, network signal, package manager hygiene — all with a single command.

Instant

Full system scan in under a second. No background processes, no setup required.

Zero privileges

Runs with standard user permissions. No root access, no launch daemons installed.

14 modules

Battery, CPU, memory, disk, security, network, packages, backups — and more. Skip what you don't need.

Pipe-friendly

Color output with --no-color fallback. Parse and log with --quiet.

Homebrew install

One command to install. Updates flow through your standard brew upgrade workflow.
